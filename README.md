# Semana Acadêmica — Oficina de Git

Repositório criado durante a oficina de Git da Semana Acadêmica para praticar versionamento de código com Git e GitHub.

---

## Tópicos aprendidos

- Inicializar um repositório com `git init`
- Adicionar arquivos com `git add .`
- Salvar alterações com `git commit -m "mensagem"`
- Enviar para o GitHub com `git push`
- Resolver conflitos de merge
- Trabalhar com branches (`master` / `main`)

---

## Comandos utilizados

```bash
git init                     # inicializa o repositório
git add .                    # adiciona todos os arquivos
git commit -m "mensagem"     # salva as alterações
git push                     # envia para o GitHub
git pull                     # baixa atualizações do GitHub
git status                   # verifica o estado dos arquivos
git remote add origin <url>  # conecta ao repositório remoto
```

---

## Dica

Sempre salvar o arquivo antes de rodar o `git add .` — caso contrário as alterações não serão registradas!
